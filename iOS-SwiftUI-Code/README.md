

## 项目简介
本项目是基于 SwiftUI 开发的 iOS 应用，旨在提供直观的用户界面和流畅的交互体验。该应用支持多种功能，包括但不限于：

- **响应式 UI 设计**：适配不同屏幕尺寸。
- **深色/浅色模式切换**：支持系统主题同步。
- **网络请求**：基于 `URLSession` 进行 API 交互。

## 环境要求
在开始使用该项目之前，请确保您的开发环境满足以下要求：

- Xcode 14.0 及以上版本
- iOS 15.0 及以上
- Swift 5.7+
